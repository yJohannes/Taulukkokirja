Määritelmä ja laskutoimitukset

Määritelmä

\mathbb{C}=\left\{z\mid z=a+bi{,}\ \ a{,}\ b\in\mathbb{R}{,}\ i^2=-1\right\}

Imaginaariyksikköä i vastaa kompleksitason piste (0, 1) .

Kompleksilukua z=a+bi vastaa kompleksitason piste (a, b) .

Nimi Lause Selite

Reaaliosa

 \text{Re }z=a

imaginaariosa

 \text{Im }z=b

identtisyys

 z_1=z_2 \Leftrightarrow \text{Re } z_1= \text{Re } z_2 \text{ ja } \text{Im } z_1= \text{Im } z_2

nimityksiä

 b \neq 0 \Rightarrow z

imaginaarinen
a=0 \text{ ja } b \neq 0 \Rightarrow z=bi puhtaasti imaginaarinen
b=0 \Rightarrow z=a reaaliluku

vastaluku

 -z=-a-bi

liittoluku eli kompleksikonjugaatti

 \bar{z}=a-bi
liittoluvun ominaisuuksia \overline{z_1+z_2}=\bar{z}_1+\bar{z}_2
\overline{z_1z_2}=\bar{z}_1\bar{z}_2
\overline{\left(\frac{1}{z}\right)}=\frac{1}{\bar{z}}
\overline{\left(\frac{z_1}{z_2}\right)}=\frac{\bar{z}_1}{\bar{z}_2}

itseisarvo eli moduuli

 \left| z \right|=\sqrt{{{a}^{2}}+{{b}^{2}}}=\sqrt{z\bar{z}}
itseisarvon ominaisuuksia z\bar{z}=\left|z\right|^2
\left|z_1z_2\right|=\left|z_1\right|\left|z_2\right|
\left|\frac{z_1}{z_2}\right|=\frac{\left|z_1\right|}{\left|z_2\right|}

käänteisluku

 {{z}^{-1}}=\dfrac{1}{z}=\dfrac{{\bar{z}}}{z\bar{z}}=\dfrac{a-bi}{{{a}^{2}}+{{b}^{2}}}
