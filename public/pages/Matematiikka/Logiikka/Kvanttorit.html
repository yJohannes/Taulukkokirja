Kvanttorit

Olemassaolo- eli eksistenssikvanttori \exists

\exists x \in A \text{: } p(x)

On olemassa (ainakin yksi) sellainen A :n vakio x , jolle pätee väite p(x)


Universaali- eli kaikkikvanttori \forall


\forall x \in A \text{: } p(x)

Kaikille A :n vakioille x pätee väite p(x)


Yhdistetyt kvanttorit

\exists x \forall y \text{: } p(x,y)

On olemassa sellainen x , että olipa y mikä tahansa, niin väite p(x, y) pätee

\forall x \exists y \text{: } p(x,y)

Jokaista x :n arvoa kohti on olemassa sellainen y , että väite p(x,y) pätee.


Kvanttorin negaatio

\neg (\exists x \in A \text{: } p(x)) \Leftrightarrow \forall x \in A \text{: } \neg p(x)
\neg (\forall x \in A \text{: } p(x)) \Leftrightarrow \exists x \in A \text{: } \neg p(x)
