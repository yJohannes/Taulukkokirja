Hyperboliset funktiot \cosh ^2x-\sinh ^2x=1 Areafunktiot (vrt. arkus (arc)) ovat käänteisfunktioita hyperbolisille funktioille, jotka on rajattu bijektioiksi. \begin{matrix} \sinh \ :\ \mathbb{R}\to \mathbb{R}\ \ \ \ \ \ \ \ \ \ \ \ \ &\operatorname{arsinh}\ :\ \mathbb{R}\to \mathbb{R}\ \ \ \ \ \ \ \ \ \ \ \ \ \\ \cosh \ :\ \left[0{,}\ \infty \right)\to \left[1{,}\ \infty \right)&\operatorname{arcosh}\ :\ \left[1{,}\ \infty \right)\to \left[0{,}\ \infty \right]\\ \tanh \ :\ \mathbb{R}\to \left(-1{,}\ 1\right)\ \ \ \ \ &\operatorname{artanh}\ :\ \left(-1{,}\ 1\right)\to \mathbb{R}\ \ \ \ \ \end{matrix} \operatorname{arsinh}x=\ln \left(x+\sqrt{x^2+1}\right) \operatorname{arcosh}x=\ln \left(x+\sqrt{x^2-1}\right) \operatorname{artanh}x=\frac{1}{2}\ln \left(\frac{1+x}{1-x}\right)
